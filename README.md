# nuxt-animate.css
A module to include animate.css in your nuxt project

# Installation

## npm
npm i @neneos/nuxt-animate.css

## yarn
yarn add @neneos/nuxt-animate.css

# Hooking it up

In the nuxt.config file add the module in the modules section

~~~~
module.exports = {
  ...
  /*
  ** Nuxt.js modules
  */
  modules: [
    '@nuxtjs/axios',
    '@neneos/nuxt-animate.css'
  ]
  ...
}
~~~~
